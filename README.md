# Pranayama Practice Tool

A simple, focused web application for guided pranayama breathing practice.

## What is it?

This tool helps you practice **Nadi Shodhana Pranayama** (alternate nostril breathing) with visual feedback and sound cues. It includes 5 preset difficulty levels and a custom mode for personalized practice.

## Features

- **5 Practice Levels** - Progressive difficulty from beginner to advanced
- **Custom Settings** - Create your own breathing rhythm
- **Visual Feedback** - Watch your nostrils and breath bar animate in real-time
- **Sound Cues** - Optional audio feedback at phase transitions
- **Dark/Light Mode** - Easy on the eyes in any lighting
- **Color Customization** - Change colors to your preference
- **Fullscreen Mode** - Minimal distractions for deep practice
- **Progress Tracking** - Monitor total practice time and cycles completed
- **Responsive Design** - Works on desktop, tablet, and mobile

## How to Use

1. Open `index.html` in your browser
2. Select a practice level from the dropdown
3. Read your nostril, then press **Start**
4. Follow the on-screen instructions and visual cues
5. Use **Pause** to take a break, **Reset** to start over

**Start with Level 1** and only advance once you can practice comfortably for a full hour.
